### 🧠 About me

Self-taught developer from **Ukraine**, on GitHub since 2020. I like taking things
apart to understand how they work — and putting them back together as something
**automated**. Most of my day-to-day is JavaScript / Node.js: reverse engineering
APIs and protocols, and building bots, parsers and small tools around them.

- 🔍 **Reverse Engineering** — analyzing apps, protocols and hidden APIs
- 🤖 **Bots & Automation** — turning repetitive work into reliable scripts
- 🕸️ **Parsers & Scraping** — collecting and structuring data at scale
- ⚙️ **Tooling** — tiny utilities that make everyday work smoother
